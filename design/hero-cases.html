칩 문법은 경우마다 자연스럽게 달라집니다.

  • 비모 일 있음 → 비모 배지(V마크) + 수정 요청 N(고스트) + 마감 D-X(주황). 가장 강한 톤.

  • 둘 다 조용 → 칩 없이 최근 승인 N + 업로드 유도. 옅은 초록.
