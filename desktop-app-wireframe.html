아키텍처

컴포넌트별 역할
파일 watcherchokidar (Node) / notify-rs (Rust). 새 파일 감지, 변경 감지, 5분 폴링 백업
SQLite 매핑 DB로컬 경로 ↔ vibox file ID. 중복 업로드 방지, 재개 지원
업로드 큐순차 1개씩, 동시 청크 24 conn. lib/upload.ts 재사용 또는 Rust 재작성
메뉴바 UI네이티브 메뉴 + popover (작은 React 앱). idle/uploading/error 시각화
자동 업데이트electron-updater / Tauri updater. 새 버전 자동 다운·재시작
인증OAuth 코드 플로우. 토큰 macOS Keychain · Win Credential Vault 보관
