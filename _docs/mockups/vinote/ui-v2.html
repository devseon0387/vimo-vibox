비노트 UI v2 — 3-pane 통합

v1 검토 결과 갤탭(S9 FE+) 3-pane 디자인을 PC에도 적용. 모바일은 v1 그대로 유지. PC에선 가로 여유 공간에 접을 수 있는 우측 도구 패널(목차·백링크·버전·태그)을 추가해 활용도를 높이되, 기본은 접혀 있어 글쓰기 몰입을 깨지 않습니다.

결정 PC = 갤탭 = 같은 3-pane 골격. 폭만 다르고 인지 구조는 동일. Rail 96px / List 320px(PC)·300px(태블릿) / Editor flex.
유지 모바일은 v1 디자인 그대로 — 하단 5탭 + 중앙 새 글 FAB.

비박스 PWA 보강 회의록

2026-05-27, vibox-mac1, 16:00–17:20. 참석: 선, 클로드. 목표는 PWA MVP 마무리와 다음 스프린트 범위 확정.

합의된 것

  • Share Target — 모바일 공유 시트에서 비노트로 텍스트/이미지 받기 동작 검증.
  • 아이콘 — 192/512px 두 벌 + maskable. 다크모드는 v0.3에서.
  • 푸시 — Service Worker push 등록까지 완료. 트리거 통합은 다음 차.
  • Background Sync — offline 큐 IndexedDB → online 회복 시 자동 flush.

남은 것

푸시 트리거 통합: [[받은편지함]] · 코멘트 · AI 검수. claim 자동화는 launchd에 cron으로 5분마다 — 이미 com.vibox.prune 패턴 있음. screenshots는 갤탭에서 캡처해서 manifest.json에 올리면 끝.

"flush 중 ETag 불일치 → conflict 모달" — 이 케이스 실제로 한 번 봤는데, 모달이 좀 무거웠음. 시안 다시.

다음 미팅 후 종합본은 옵시디언 [[비박스/스프린트]] 로.
