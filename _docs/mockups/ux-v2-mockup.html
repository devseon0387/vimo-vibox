7. 로딩 Skeleton spinner 대체

초기 로드 / 페이지 전환 시 spinner 대신 layout placeholder. 실제 콘텐츠 도착 전까지 회색 박스가 자리잡고 있어 perceived speed +30%.

shimmer 애니메이션 — 좌→우 광택 흐름
실제 layout 그대로 — 콘텐츠 도착 시 jump 없음
최소 100ms 표시 — 빠른 응답엔 spinner 대신 그냥 표시 안 함 (flash 방지)
