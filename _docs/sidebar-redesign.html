① "렌더링" → "공용 폴더"
직관적 이름. "팀 공용 영상 작업 폴더"라는 본질이 즉시 와닿음 RENAME
② "인사이트" → "검수 통계" + 위치 이동
"도구"가 아닌 VIMO Box 섹션 안으로. 팀 검수 결과니까 자연스러움 RENAME MOVE
③ "내 기록" 새로 노출
이전엔 사이드바에 없어서 프로필 영역으로 가야 도달. Personal 섹션에 정착 ADD
④ "도구" → "일상" + 인사이트 분리
섹션 의미를 더 분명히. 인사이트가 빠져나가서 깔끔 RENAME
⑤ 관리 섹션 빈도 순 재정렬
트래픽 통계 → 저장소 정리 → 사용자 관리 (자주 쓰는 순) REORDER
⑥ 활성 표시 강화
왼쪽에 accent 세로 바 4px. 폴더 깊이 들어가도 위치 인지 즉시 VISUAL
⑦ NEW 배지
신기능 항목 옆에 7일간 배지 (localStorage로 사용자별 추적) DISCOVERY
⑧ 시스템 미니 위젯
인코딩 큐·백업 상태·디스크 잔여 한 줄 요약. 상시 모니터링 NEW
⑨ 프로필 클릭 → 드롭다운
이메일 노출 + 클릭 시 메뉴 (내 박스 / 내 기록 / 설정 / 로그아웃) UX
